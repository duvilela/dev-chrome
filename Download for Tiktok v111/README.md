# Download for Tiktok (v1.1.1)

O **Download for Tiktok** é uma extensão de navegador leve, rápida e totalmente livre de anúncios, desenvolvida para salvar vídeos e fotos do TikTok diretamente no seu computador sem qualquer marca d'água e sem exigir login.

Esta versão (**v1.1.1**) é distribuída de forma limpa, segura e sem propagandas.

---

## ✨ Principais Características

- **Download sem Marca d'Água**: Salve os vídeos do TikTok em qualidade original (MP4) livres da marca d'água oficial.
- **Download em Massa**: Baixe de uma só vez vários vídeos de uma página de perfil de usuário ou explore.
- **Suporte Integrado a Inteligência Artificial**: Botão de integração direta com o ChatGPT (GPT-4) para analisar, transcrever ou resumir o conteúdo do vídeo a partir da sua URL.
- **Instalação Descompactada Segura**: Sem anúncios embutidos, sem rastreadores e sem coleta de dados pessoais.

---

## 🛠️ Como Instalar no Google Chrome (Modo Desenvolvedor)

Como a extensão é distribuída no formato de código aberto descompactado, siga os passos abaixo para instalar:

1. Faça o download ou clone a pasta `Download for Tiktok v111` do repositório para o seu computador.
2. Abra o Google Chrome e digite **`chrome://extensions/`** na barra de endereços.
3. No canto superior direito da página de extensões, ative a opção **"Modo do desenvolvedor"**.
4. No canto superior esquerdo, clique no botão **"Carregar sem compactação"**.
5. Localize e selecione a pasta **`Download for Tiktok v111`** (a pasta principal que contém o arquivo `manifest.json`).
6. A extensão será instalada e estará visível na barra de ferramentas do seu navegador!

---

## 🚀 Como Usar

1. **Baixar Vídeo Único**:
   - Abra o TikTok no navegador ([tiktok.com](https://www.tiktok.com)).
   - Ao passar o mouse sobre os vídeos no feed ou ao abrir a página de um vídeo específico, você verá um botão de download (azul/verde) adicionado ao painel do vídeo.
   - Clique nele para baixar o vídeo diretamente sem marca d'água.
2. **Integração com ChatGPT (IA)**:
   - Ao lado do botão de download de vídeo, você verá um botão com o ícone **"AI"**.
   - Clique nele para abrir automaticamente o ChatGPT com um prompt pré-configurado contendo o link do vídeo, permitindo solicitar resumos ou transcrições ao GPT-4.
3. **Downloads em Massa**:
   - Acesse o perfil de qualquer criador de conteúdo no TikTok. Use os botões de download em lote para baixar múltiplos vídeos simultaneamente.
